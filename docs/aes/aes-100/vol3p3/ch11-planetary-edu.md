# AES-100 Vol III P3 Ch 11 — Planetary Educational Intelligence Network (public/aquin-planetary-edu.js)

Federated (not centralized) global education; distinct from generic federation
(aquin-civilization.js) + credentials (aquin-identity.js). Node-tested (5).
Product-relevant (EduRankAI = credentials/education).
- **Sovereign credential recognition**: institution B recognises A's credential only
  via a recognition agreement OR shared accreditation body — federation never forces
  recognition (each institution's policy decides).
- **Cross-system equivalence**: grades/credits normalised through a quality scale —
  GPA 3.6/4 → 90% → 90 ECTS; 3 US credits → 6 ECTS / 30 UK.
- **Learning mobility**: a moving learner transfers only recognised credentials,
  converted to the destination's system (TU-Delft rejects unrecognised MIT).
HONEST SCOPE: recognition/equivalence/mobility real; credential signing
(aquin-identity.js), inter-governmental accreditation treaties declared substrates.
